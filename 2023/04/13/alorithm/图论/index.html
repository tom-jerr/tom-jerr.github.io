图论——算法

本文最后更新于：2023年4月13日 晚上

前言

图论

BFS

  • 剪枝：dist不更新的点(dist[nx][ny] <= dist[cx][cy])，直接break，不再进行搜索

方格迷宫

  • 搜索空间过大，需要剪枝
  • 将dist不更新的点(dist[nx][ny] <= dist[cx][cy])，直接break，不再进行搜索

拓扑序列

模板

  • 计算有向图的度
  • 如果度为0就入队；将节点去掉，将其所影响的节点的度-1；重复上述操作

构造有向无环图

  • 最后的拓扑序列一定是从小的指向大的；所以记录无向边端点再拓扑排序中的位置
  • 如果前面的有向边不能成为拓扑排序，说明加入无向边后也不能

Dijkstra

点稠密型

  • 使用邻接矩阵

堆优化版Dijkstra

  • 适合点稀疏

超级源点和超级汇点

  • 超级源点跟超级汇点是模拟出来的虚拟点，多用于图中：
    <1>同时有多个源点和多个汇点，建立超级源点和超级汇点

    <2>同时有多个源点和一个汇点，建立超级源点

    <3>同时有多个汇点和一个源点，建立超级汇点

最短路径

SPFA

  • 存在负权边时使用
  • 通过一个cnt计数数组可以判断是否存在环

SPFA判断负环

  • 必须将每个点都放入队列进行判断
  • dist（距离）初值定义为0

最优贸易

  • 两次SPFA分别计算最小值和最大值

作物杂交

  • 在构建图的过程中，边不仅可以存储边权，还可以存储其他额外信息

  • 把所有初始的作物加到队列中；将能够生成的作物加到图中

Floyd

  • 计算连通块之间的最大值的最小值使用floyd（n较小时）

牛的旅行

  • 转换成两个连通块求最大距离的最小值
  • maxd[i]求出与 i 连通的最大值
  • 更新最大值的最小值res2 = min(res2, getdist(i, j) + maxd[i] + maxd[j] )

最小生成树

Prime算法

  • 每次选择最短的点加入生成树

Kruskal

  • 每次加入最短边；并查集
