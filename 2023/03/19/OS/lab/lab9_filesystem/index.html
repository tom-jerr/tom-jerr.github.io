前言

本文是对MIT6.S081中lab9 file system的实现

Lab9 File system

任务要求

为xv6的文件系统增添一些功能：inode 的双重间接块及符号链接

功能实现

Large files

思路

  • 在本部分中需要为 inode 提供双重间接块，使得xv6中的最大文件大小被增加。其原理与 inode 中的原有间接块类似：在双重间接块中，存放了 NINDIRECT 个间接块，而一个间接块中又存放了 NINDIRECT 个直接磁盘块，因此在双重间接块中即可表示 NINDIRECT*NINDIRECT 个直接磁盘块，其使得xv6的最大文件大小大大增加。

  • kernel/fs.h

  • kernel/file.h

  • kernel/fs.c修改bmap & itrunc

Symbolic links(符号链接)

思路

  • 添加系统调用

  • kernel/syscall.c完成主体函数sys_symlink

  • 修改sys_open函数，增加对软链接的判断

  • 增加标志位

    • kernel/stat.h中添加T_SIMLINK
    • kernel/fcntl.h中添加O_NOFOLLOW
