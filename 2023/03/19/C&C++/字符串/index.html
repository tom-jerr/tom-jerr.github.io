C - String

本文最后更新于：2023年3月19日 晚上

前言

  • 对C的字符串进行部分说明

字符串

字符串数组

  • 大小要比字符串长度至少多1（容纳空字符）
  • 使用指针指向字符串进行内容修改；可能导致所有使用该字符串的内容均修改

gets()

  • 唯一参数words，只存储数组的首地址
  • 丢弃换行符
  • 不安全，容易造成缓冲区溢出

fgets()

  • 从键盘输入，第三个参数是 stdin
  • 会存储换行符

gets_s()

  • 只从stdin中读取输入
  • 读到换行符丢弃换行符

scanf()

  • 使用%s转换得到字符串
  • 遇到空白字符即停止（空行、空格、制表符、换行符）

puts()

  • 遇到空字符，输出字符串

fputs()

  • 第二个参数指明要写入的文件

字符串函数

  • ANSI C标准将这些函数原型放在$string.h$中

strlen()

  • 计算字符串的长度，不计算末尾空字符

strcat() & strncat()

  • 将第二个字符串备份后，与第一个字符串拼接；返回第一个字符串
  • 增加了最大追加字符长度n

strcmp()

  • 如果返回值小于 0，则表示 str1 小于 str2。
  • 如果返回值大于 0，则表示 str1 大于 str2。
  • 如果返回值等于 0，则表示 str1 等于 str2。

strcpy() & strncpy()

  • 将src复制到dest中，返回dest
  • 最多复制n个字符

sprintf()

  • str – 这是指向一个字符数组的指针，该数组存储了 C 字符串。

  • format – 这是字符串，包含了要被写入到字符串 str 的文本。它可以包含嵌入的 format 标签，format 标签可被随后的附加参数中指定的值替换，并按需求进行格式化。：

  • 如果成功，则返回写入的字符总数，不包括字符串追加在字符串末尾的空字符。如果失败，则返回一个负数。
