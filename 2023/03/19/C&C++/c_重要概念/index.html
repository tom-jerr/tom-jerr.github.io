C - 重要概念

本文最后更新于：2023年5月15日 晚上

前言

  • 对C的部分重要概念进行部分说明

函数

字符串转换成整数

strstr()

  • 找到子串在一个字符串中第一次出现的位置,在string.h头文件中

strchr()

  • 找到一个无符号字符c在字符串中首次出现的位置，返回char*类型
  • 指针相减：返回的是类型的个数，这里是出现字符距离字符串开始的长度

abort()

  • 中止程序执行，直接从调用的地方跳出。

atexit()

  • 当程序正常终止时，调用指定的函数 func。您可以在任何地方注册你的终止函数，但它会在程序终止的时候被调用。

exit()

  • 立即终止调用进程。任何属于该进程的打开的文件描述符都会被关闭，该进程的子进程由进程 1 继承，初始化，且会向父进程发送一个 SIGCHLD 信号。

存储

extern

  • 外部存储类别
  • 若变量被声明在另一个文件中，必须使用extern

分配内存

malloc()

  • size – 内存块的大小，以字节为单位。

  • size – 内存块的大小，以字节为单位。

calloc()

  • nitems – 要被分配的元素个数。

  • size – 元素的大小。

  • 该函数返回一个指针，指向已分配的内存。如果请求失败，则返回 NULL。

  • 将所有内存初始化为0

词法注意

“贪心法”

  • 编译器将程序分解成符号，是从左到右依次分解
  • 除了字符串、字符常量，符号中间不能有空白

字符与字符串

  • 单引号引起的字符实际上代表一个整数
  • 用双引号引起的字符串，代表的是一个指向无名数值起始字符的指针；最后有一个额外的二进制字符’\0’
  • ‘yes’大多数编译器认为是一个整数值；”yes”是一个连续内存单元的首地址

运算符

  • 逻辑运算符低于关系运算符
  • 移位运算符比算术运算符要低，比关系运算符要高
  • ==和!=的优先级要更低在关系运算符中
  • =号赋值运算符从右向左结合
