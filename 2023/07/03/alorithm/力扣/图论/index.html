连通分量问题

  • DFS

DFS

通知所有员工的时间

  • 整个可以构造成一个树的结构；进行递归

  • 自顶向下递归

  • 自底向上递归：由叶子节点开始递归

BFS

颜色交替的最短路径

  • 双源BFS求解：使用广度优先搜索获取从节点 0 到某一节点的两种类型的颜色交替最短路径的长度，广度优先搜索的队列元素由节点编号和节点路径类型组成

拓扑排序

  • 拓扑排序是找到图中入度为 0 的节点，以及仅由入度为 0 节点所指向的节点

找到最终的安全状态

  • 找到出度为0以及仅由出度为0节点所指向的节点
  • 利用拓扑排序

课程表

  • 拓扑排序，判断是否所有课程都可以访问

  • 队列出队时记录访问的元素个数

判断图的入度与出度（涉及到二维矩阵和指向问题）

找到小镇的法官

  • 题干描述了一个有向图。每个人是图的节点，我们可以遍历 trust统计每个节点的入度和出度，存储在inDegrees和 outDegrees中。
  • 根据题意，在法官存在的情况下，法官不相信任何人，每个人（除了法官外）都信任法官，且只有一名法官。因此法官这个节点的入度是n-1，出度是0
