树

二叉树

判断是否是完全二叉树

  • 使用层次遍历
  • 将所有节点，包括空节点入队列；如果空节点后面还有节点说明不是完全二叉树

求二叉树高度

  • 使用层次遍历，last记录上层的最右节点，之后跳转到下层的最右节点
  • 通过last与队列尾部rear实现找到每层的最右节点

先序序列和中序序列确定二叉树

  • 使用递归方式确定，将两个序列以及起始位置传递进去

寻找data=x节点的所有祖先

  • 使用非递归后序遍历，因为根节点最后访问，所有栈中所有元素均是该节点祖先

寻找p，q两个节点的公共祖先

  • LCA问题：倍增法、Tarjan法
  • 使用后序遍历和两个栈，通过比较栈中是否有祖先相等来判断

Morris中序遍历

  • 每个节点遍历两次
  • Morris 遍历算法整体步骤如下（假设当前遍历到的节点为 x）
    • 如果 x 无左孩子，则访问 x 的右孩子，即 x=x.right。
    • 如果 x 有左孩子，则找到 x 左子树上最右的节点（即左子树中序遍历的最后一个节点，x 在中序遍历中的前驱节点），我们记为predecessor
    • 根据 predecessor 的右孩子是否为空，进行如下操作
      • 如果 predecessor 的右孩子为空，则将其右孩子指向 x，然后访问 x 的左孩子，即 x=x.left。
      • 如果 predecessor 的右孩子不为空，则此时其右孩子指向 x，说明我们已经遍历完 x 的左子树，我们将 predecessor 的右孩子置空，然后访问 x 的右孩子，即 x=x.right。

恢复二叉搜索树

  • 使用pre进行存储上一个结点

  • Morris中序遍历

  • 使用两个指针寻找逆序的节点

构造不同的二叉搜索树

  • 选定不同的根节点；不断递归寻找左右子树
  • 将左右子树随机拼接

判断是否是平衡二叉树

自底向上

  • 如果左右子树的高度差大于1，返回-1
  • 自底向上递归，如果左子树或者右子树返回-1.提前结束递归

自顶向下

  • 如果左右子树的高度差小于1，同时左右子树均平衡；该树是平衡二叉树

二叉树最小深度

  • 递归子树

  • 使用广度优先搜索，保证首先搜到的一定是深度最小的节点
