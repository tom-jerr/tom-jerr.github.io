# Visual-Inertial SLAM as Simple as A, B, VINS

## Abstract

- 使用 3 个深度神经网络进行密集深度图的提取
- 使用紧凑的 AB 特征表示并利用多视图信息进行校正
- 使用 memory tree 数据结构来存储和管理关键帧，减少 loop closure 对更多状态的影响

## Intro 总体介绍

- 提出了一种称为 AB 特征的紧凑特征表示，其中任意数量的特征位置由 a 和 b、单目深度图的尺度和偏置以及其他可选项来参数化，以利用多视图信息来校正深度。
- 引入了一种新的数据结构- -内存树，用于显著加速位姿图优化。记忆树允许 AB - VINS 解决位姿图 SLAM，同时只重新线性化恒定数量的变量，

## 背景知识的相关工作

与 AB VINS 工作类似的工作一般具有 VIO 前端和基于优化的后端，旨在产生与真实估计误差一致的协方差的状态估计。

这些方法都存在该问题：受全局优化影响的变量个数随关键帧数量呈线性增长，而**AB - VINS 可以在只重新线性化常量变量的情况下求解闭环**。

在基于优化的系统中，如果计算边缘协方差，与 EKF 相比，边缘化的计算成本很高，这促使我们选择在 AB - VINS 中消**除边缘化**，而不是像 ORB - SLAM 那样简单地在滑动窗口中固定最古老的位姿。
